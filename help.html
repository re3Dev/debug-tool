Re:Bugger Help Page

Ping the Printer Connection
Checks to see if the printer is properly connected via Ethernet.
Grab Logs
Executes a script to retrieve a Klippy log from the printer.
Interface Pop-up
Opens the browser in to display the printer's mainsail interface.
Generate Load Graph
Runs a script to generate a load graph from a selected log file and then displays it.
Display Last Loadgraph
Opens the browser and displays the latest generated load graph.
Display Errors in Log
Parses the log file and finds specified errors with line numbers.
Display Extruder Temps
Shows the last recorded temperatures of the all the extruders and heater bed from the printer's logs
Manual GCode
Sends a specific user written GCode to the printer connected.
Flash IMG to SD Card
Flashes the pre-loaded firmware to the connected SD Card in the USB slot.
Export Logs & Graphs
Takes all the logs and graphs created and exports them to an attached USB Drive.
Bluetooth Connect
Connects the keyboard attached to re:Bugger using bluetooth.
Terminal Popup
Uses "Ctrl+Alt+F1" to bring up the terminal, to exit use "Ctrl+Alt+Fn+7"
Delete All Logs & Graphs
Deletes all the saved logs and any previously stored graphs.
